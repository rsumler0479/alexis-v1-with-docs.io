Text colors

You can use predefined class to change the color of text. 


Example:

<p class="red-text">Michael Schmitt</p>

 

List of classes:

<p class="red-text">For red text</p>

<p class="green-text">For green text</p>

<p class="puprle-text">For purple text</p>

<p class="yellow-text">For yellow text</p>

<p class="dark-text">For dark text</p>

<p class="white-text">For white text</p>

 

NOTE: This classes will change the icons color too.

Subscribe Form

Subscribe form will save emails to MailChimp list.

 

How to connect mail chimp list?

Open custom.js and find the following script. Just replace that MailChimp form action URL with your MailChimp list URL

 

How to find mailchimp form action URL?

Just go to your MailChimp account. Then your list. Then Signup forms. Select Embedded forms You will find form action URL in this page.

Contact Form

The contact form will use PHP to send emails. Change the current email spambox@designlab.co  to your email.

 

1. I used PHP mail in Contact Form to send emails. Maximum time it will go to spam/junk folder. Please check your spam/junk folder.

2. If you use the custom email address like yourname@yourcompany.com. Custom mail apps block PHP mail. So use the email address with Gmail, yahoo, etc.

3. Make sure your server allowing / supporting sendmail.php to send the email. Sometimes the server will delay the mail delivery.
